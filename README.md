# Calculadora de Gorjeta
Sabe quando você vai com os amigos para um bar ou uma balada e tem 6/7 pessoas e a conta deu R$572,50 e todo mundo pega o celular? Essa é uma aplicação que faz esse cálculo. O usuário digita o valor, a quantidade de pessoas e o quanto pode dar de gorjeta. A aplicação faz o cálculo total.
